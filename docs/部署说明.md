# 后端服务主机部署

1. 构建镜像：`docker build -t mall-backend:<tag> .`
2. 推送镜像到镜像仓库，并在主机设置 `MALL_BACKEND_IMAGE`。
3. 将生产配置写入 Etcd：`/configs/mall/system`，内容参考 `mall_local.yml.example`。
4. 确认 MySQL 已执行需求文档中的 DDL，并使用 `make gendb` 生成模型。
5. 启动服务：`docker compose -f docker-compose.prod.yml up -d`。
6. 验收：访问 `/ping`，确认 Redis、MySQL、Etcd 正常；微信支付和退款回调地址必须是 HTTPS 公网地址。
